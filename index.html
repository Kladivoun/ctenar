Čtenáři

Přejmenování ani smazání čtenáře nesmaže data ostatních.

    Import z Google tabulky

    Vlož odkaz na publikovanou tabulku ve formátu CSV. V Google Sheets: Soubor → Sdílet → Publikovat na web → vyber list → formát CSV → zkopíruj odkaz sem. Import knihy přidá nebo aktualizuje podle autora+názvu; tvůj tracking (čtečka, přečteno, hodnocení, poznámky) zůstane zachován.

    Obnova ze zálohy (JSON)

    Vyber JSON zálohu vytvořenou tlačítkem Export JSON.
    Přepsat = nahradí veškerá aktuální data obsahem zálohy (nevratné).
    Sloučit = doplní a aktualizuje knihy podle autora+názvu; existující tracking (čtečka, přečteno, hodnocení, poznámky) zůstane zachován a prázdná pole v záloze nic nepřepíšou.

    Synchronizace (GitHub Gist)

    Data se zrcadlí do soukromého GitHub Gistu, takže je máš na všech zařízeních. Token i Gist ID se ukládají jen v tomto prohlížeči (localStorage), nikam se neposílají. Token je citlivý údaj – chovej se k němu jako k heslu.
